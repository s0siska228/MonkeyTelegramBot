# 🐒 MonkeyBot — Личный учет расходов

Учебный проект (Backend, 1 курс) по созданию Telegram-бота для быстрого финансового учета. Бот позволяет записывать траты «на лету» и получать моментальные отчеты.

## 🚀 Основные возможности

* **Быстрая запись:** Просто отправьте сообщение вида `500 Кофе`.
* **Отчеты:** Просмотр суммарных трат за сегодня, неделю или месяц.
* **История:** Вывод последних 10 операций с датой и временем.
* **Часовой пояс:** Бот работает по московскому времени (**МСК/UTC+3**).
* **Надежность:** Все данные сохраняются локально в формате JSON.

## 🛠 Технологии

* **Язык:** C# (.NET 8+)
* **Библиотека:** `Telegram.Bot`
* **Хранение данных:** JSON-файл (`expenses.json`)
* **Парсинг:** `System.Text.Json`

## 📋 Команды бота

| Команда | Описание |
| :--- | :--- |
| `/start` | Краткая инструкция и приветствие. |
| `/today` | Сумма трат за текущие сутки (с 00:00 МСК). |
| `/week` | Сумма трат за последние 7 дней. |
| `/month` | Сумма трат за последние 30 дней. |
| `/last` | Список из 10 последних записей. |

**Формат добавления траты:** `<сумма> <описание>`  
*Пример: `1200 такси в офис`*

## 💻 Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone [https://github.com/s0siska228/MonkeyTelegramBot.git](https://github.com/s0siska228/MonkeyTelegramBot.git)
    ```

2.  **Настройте токен:** Замените `XXX` в коде (файл `Program.cs`) на ваш актуальный токен от BotFather.

3.  **Запустите проект:**
    ```bash
    dotnet run
    ```
